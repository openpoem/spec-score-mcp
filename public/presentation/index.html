Spec Score MCP

A balanced spec produces balanced code. An unbalanced spec produces creative fiction.

completeness — Can the LLM understand the full scope?  ·  clarity — Is there only one way to interpret this?
constraints — Does the LLM know what NOT to build?  ·  specificity — Are there concrete, testable details?
